Git is a free & open source, distributed version control system designed to handle everything from small to very large projects with speed and efficiency.

Git is easy to learn, has lightening fast performance and a tiny footprint. It outclasses SCM tools like Mercurial, Subversion, CVS, Perforce, and Team Foundation Server with features like cheap local branching, distributed workflows, and convenient staging areas.
